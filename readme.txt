== Description ==

This plugin is add-on for [rtPanel Theme Framework](http://wordpress.org/extend/themes/rtpanel "rtPanel Theme Framework") and should be used alongwith it.

It adds an option page under rtPanel theme options using which a user can write codes for various actions/filters hooks supported by rtPanel Theme Framework - [http://rtpanel.com/docs/developer/](http://rtpanel.com/docs/developer/ "rtPanel Hooks").

Using this plugin a user need not edit any theme files to achieve desired functionality.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
